# DevNote – MERN Notes Application

DevNote is a full-stack notes application built using the **MERN stack**.

It allows users to create, edit, search and delete notes through a clean and minimal interface.

---

## Features

• Create notes
• Edit notes
• Save notes manually
• Delete notes
• Search notes
• Modern UI layout

---

## Tech Stack

Frontend
• React
• CSS
• Axios

Backend
• Node.js
• Express

## Run Locally

### Backend

cd backend
npm install
npm start

Server runs on

http://localhost:5000

### Frontend

cd frontend
npm install
npm start

Frontend runs on

http://localhost:3000
